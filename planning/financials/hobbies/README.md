# Hobbies Planning

## Overview

This section tracks hobby expenses with a target of keeping each hobby under 1% of after-tax income. This ensures hobbies remain sustainable and don't compromise financial security.

## Hobbies

| Hobby | Annual Cost | 1% Threshold Income Required |
|-------|-------------|------------------------------|
| [Sports Cars](./sports-cars.md) | $24,000-$44,400 | $2.4M-$4.44M after-tax |
| [Video Game Development](./video-game-development.md) | $5,000-$10,000 | $500K-$1M after-tax |
| [Warhammer Armies](./warhammer.md) | ~$2,000/yr (ongoing) | $200K after-tax |
