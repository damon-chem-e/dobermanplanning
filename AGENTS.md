## Agent Instructions

### When Creating New Files
1. **Always include references**: Every cost estimate must cite a source
2. **Use clear headers**: Structure with H1, H2, H3 for easy parsing
3. **Separate concerns**: Keep different cost types in separate files
4. **Include assumptions**: Document all assumptions made in calculations
5. **Use tables**: Present data in tables for clarity

### When Updating Files
1. **Preserve references**: Don't remove existing citations
2. **Add new sources**: Include new data sources when updating estimates
3. **Maintain structure**: Follow existing formatting patterns
4. **Document changes**: Note what was updated and why

### Cost Estimation Guidelines

#### Upfront Costs
- **Down Payment**: Typically 20% of purchase price
- **Renovation**: Based on square footage and scope
- **Closing Costs**: 2-5% of purchase price
- **Inspection**: Multiple inspections recommended (general, structural, pest, etc.)

#### Monthly Costs
- **Mortgage**: Principal + interest based on loan terms
- **Property Tax**: Based on county assessment rates
- **Utilities**: Electricity, water, gas, internet
- **Maintenance**: 1-2% of property value annually
- **Insurance**: Homeowners insurance + flood insurance if needed

#### Additional Expenses
- **Transportation**: Car loan, fuel, insurance, maintenance
- **Food**: Groceries, dining out
- **Healthcare**: Insurance, co-pays, medications
- **Entertainment**: Internet, streaming, hobbies

## Quality Standards

### Data Accuracy
- Verify all numbers with at least two sources when possible
- Use recent data (within last 12 months)
- Document any outdated information clearly

### Formatting
- Use markdown tables for data presentation
- Include clear headers and subheaders
- Use bullet points for lists and explanations
- Include currency symbols and units consistently

### Documentation
- Every file should have a clear purpose statement
- All assumptions must be documented
- Sources should be cited inline and in a references section

## Workflow

1. **Research**: Gather data from multiple sources
2. **Estimate**: Create initial cost estimates
3. **Document**: Write markdown with clear structure and references
4. **Review**: Verify calculations and sources
5. **Update**: Regularly refresh data as needed

## Emergency Procedures

If you encounter conflicting data:
1. Note the discrepancy in the documentation
2. Use the most conservative estimate
3. Document which source was chosen and why
4. Flag the file for future review

If you can't find reliable data:
1. Document the search attempts
2. Use industry standard estimates with clear caveats
3. Note the uncertainty level
4. Plan to revisit when better data is available
